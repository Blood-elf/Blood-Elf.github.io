# Vue 组件化设计开发

  • 能掌握 CSS 模块化技术以及模块化设计

  • 组件化的概念并不陌生，如何在一个复杂的项目中把组件化设计的灵活才重要

  • 早期就是插件

    • 一般都是 JavaScript
    • 无法做到彻底的组件化
    • 例如样式之类的还需要单独的导入
    • 依赖 DOM
    • 使用麻烦

  • 后来的 Angular、React、Vue

  • Web 标准：WebComponents

  • 当前主流方式：组件化开发

目前的组件化存在的问题：

  • 缺乏方法论
  • 没有方法论导致组件设计不合理，项目难以把控
  • 组件划分过细，组件数量大，当项目复杂度上去以后，整体规模难以把控
  • 组件划分太粗，项目的抽象设计以及复用性设计达不到理想的效果

组件如何合理的设计是一个非常重要的话题。

最终目标都是：

  • 开发效率高

  • 可维护性好

  • 组件化的思维方式及项目设计

  • 组件化和模块化不仅仅是 JavaScript，也包括 css

  • 如何将项目做出亮点，面试又该如何表现

  • 学会独立构建一个项目，做到独当一面
